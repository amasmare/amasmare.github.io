# 🌐 مختبرات «المختصر المفيد في شبكات الكمبيوتر» التفاعلية

> **12 مختبراً تفاعلياً بالعربية (RTL)** مبنية على **أجزاء الكتاب الفعلية** (الأجزاء العشرة + التدريب العملي).
> المرجع: **المختصر المفيد في شبكات الكمبيوتر — م. محمود يوسف أسعد** · إعداد: **م. عبدالله الأسمري**

---

## 📖 عن المشروع
ملفات HTML **مستقلة بالكامل** (CSS وJavaScript مضمّنان داخل كل ملف) — تعمل بدون إنترنت وبدون أي ملفات خارجية. كل مختبر = 4 تبويبات:
**مطابقة** · **تصنيف** · **سيناريوهات** · **اختبار** (+ **حاسبة Subnet** في الجزء العاشر). النتائج تُحفظ عبر `localStorage`.
